亿忆网提醒大家严防租房中的诈骗行为以及需要注意的事项

初步找房：

没有见面之前、看房之前任何个人资料或金钱都不要通过邮件给“房东”，无论他把自己的房子说得多么有吸引力。这类“房东”很有可能是在盗用你身份或金钱。那些无法给出电话的“房东”是因为自己根本不在澳洲，以邮件的方式进行欺诈。

签合同：

与私人房东租房也一定要签合同。细读一切条款，避免以后的纷争。双方各自保留一份合同。

押金是租房必需，2-4周因人而异。租金一般也要提前给1-2周。入住前检查房子的耗损/破坏情况并双方要书面达成共识。只要不是人为的破坏，租房期间正常的耗损房客是不需要负责的，房东更加不能从押金里扣除。

合同里记得要附加额外费用（水费电费煤气费等）是否包含在房租里。

房租：

以合同里签订的时间（周/两周/月）和价格支付。保留所有房租和押金收据。

退房：

若房子在没有人为破坏的情况下，以原来的样子交还，房东是无权扣除你的押金的。如果有任何纠纷，不要私下解决，交给Fair Trading做仲裁。
